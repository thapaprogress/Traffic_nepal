# Screenshots

Place dashboard screenshots here so they render in the main README:

| Filename | What to capture |
|---|---|
| `live_feed.png`   | The Live Feed tab with detection boxes + plate labels |
| `violations.png`  | The Violations tab (log + watchlist) |
| `analytics.png`   | The Analytics tab (charts) |
| `plates.png`      | A close-up showing a recognized license plate |

### How to capture
1. Run the dashboard: `streamlit run app.py`
2. Start detection on a traffic video
3. Use your OS screenshot tool (Win+Shift+S on Windows) to capture each tab
4. Save the images here with the exact filenames above
